# ROADFXTech RAG Service

This directory contains all files related to the RAG Service (Retrieval-Augmented Generation - Document & File Management).

## Files in this directory:
- `rag_service_schema.sql` - Database schema for the RAG Service
- `ROADFXRAG_Service_Documentation.yaml` - OpenAPI specification for RAG Service endpoints

## Service Overview:
- **Port**: 8082
- **Database**: `roadfx_rag_service`
- **Primary Focus**: Document processing and file management for RAG operations
- **Key Responsibilities**:
  - File upload and storage management for RAG operations
  - Document processing and content extraction for knowledge retrieval
  - Vector embedding generation and storage for semantic search
  - Content indexing and search capabilities for AI agents
  - Document chunk management for context retrieval
